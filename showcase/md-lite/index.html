MD-Lite

Project Overview

MD-Lite is a hyper-lightweight, minimalist Markdown editor built for speed and focus. It provides a seamless WYSIWYG experience while maintaining the simplicity of plain text — no split-pane preview needed.

View Project

Tech Stack

Tauri v2 Svelte 5 Tailwind CSS v4 Tiptap TypeScript Rust

Key Features

  • WYSIWYG Markdown Editing — real-time formatting without a separate preview pane
  • Native Experience — lightweight desktop application powered by Tauri
  • Rich Text Support — bold, italic, lists, task items, tables, and code blocks with syntax highlighting
  • Custom Title Bar — clean, distraction-free interface tailored for macOS
  • Keyboard Shortcuts — common editor shortcuts for lightning-fast editing

Impact

Redefines the Markdown editing experience by eliminating the traditional split-pane workflow. Writers, developers, and note-takers gain a native desktop tool that combines the immediacy of WYSIWYG formatting with the portability of plain-text Markdown — all in a sub-5 MB binary powered by Tauri, delivering near-instant startup and minimal resource footprint compared to Electron-based alternatives.
